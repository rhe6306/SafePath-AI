# SafePath AI

SafePath AI is an AI-powered safety-aware navigation system that enhances traditional routing by predicting safety risks along routes.

## Features

- AI-based route risk prediction
- Random Forest safety model
- Segment-level risk visualization
- Crime heatmap overlays
- Safety analytics dashboard
- AI route recommendation system

## Tech Stack

- Python
- Streamlit
- Scikit-learn
- Plotly
- Folium
- OpenStreetMap APIs

## Use Cases

- Safer commuting routes
- Night travel planning
- Smart city safety insights
- Urban risk analytics
